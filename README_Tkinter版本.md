# PDF 试题模拟考试器 - Tkinter 桌面版

## 🎯 特点

✅ **无需浏览器** - 纯桌面应用，不依赖浏览器  
✅ **双解析模式** - 支持本地快速解析和 AI 智能解析  
✅ **完整功能** - 计时、答题、成绩统计  
✅ **用户友好** - 图形化界面，操作简单  
✅ **实时日志** - 显示详细的处理过程  

## 📦 安装依赖

```bash
pip install PyPDF2 openai
```

注意：tkinter 通常随 Python 一起安装，无需单独安装。

## 🚀 使用方法

### 1. 启动程序

```bash
python exam_app.py
```

### 2. 操作步骤

1. **选择 PDF 文件**
   - 点击"📂 选择 PDF"按钮
   - 选择您的考试题目 PDF

2. **解析题目**
   - **本地解析（快速）**：使用正则表达式，速度快（几秒），适合格式规范的 PDF
   - **AI解析（推荐）**：使用 DeepSeek，准确率高，需要 1-3 分钟

3. **答题**
   - 浏览题目，选择答案
   - 可以通过按钮或左侧题目列表切换题目
   - 实时显示答题进度

4. **提交试卷**
   - 点击"📝 提交试卷"按钮
   - 查看成绩和正确答案

## 🎨 界面说明

### 顶部控制栏
- **文件选择**：显示当前选择的 PDF 文件
- **解析按钮**：本地解析或 AI 解析
- **重置按钮**：清空所有数据重新开始
- **考试时长**：设置考试时间（默认 90 分钟）
- **倒计时**：显示剩余时间

### 左侧主区域
- **题目内容**：显示当前题目的题干
- **选项**：单选按钮选择答案
- **导航按钮**：上一题、下一题、提交试卷
- **答题进度**：显示已答题数和百分比

### 右侧辅助区域
- **题目列表**：快速跳转到任意题目
- **日志窗口**：显示实时处理过程和状态

## 📋 功能特性

### 1. 本地解析
- ✅ 速度快（几秒内完成）
- ✅ 不需要网络连接
- ⚠️ 准确率取决于 PDF 格式
- ⚠️ 可能无法识别答案

### 2. AI 解析
- ✅ 准确率高
- ✅ 能识别复杂格式
- ✅ 自动提取答案
- ⚠️ 需要网络连接
- ⚠️ 处理时间较长（1-3 分钟）

### 3. 考试功能
- ⏱️ **计时器**：自动倒计时，时间到自动提交
- 📊 **进度追踪**：实时显示答题进度
- 🔄 **随意切换**：可以自由跳转到任意题目
- ✅ **答案核对**：提交后显示正确答案
- 📈 **成绩统计**：显示详细的答题统计

## 🐛 常见问题

### Q: 程序启动失败
**A:** 检查是否安装了 tkinter：
```bash
python -c "import tkinter; print('tkinter 可用')"
```

### Q: AI 解析失败
**A:** 可能的原因：
1. 网络连接问题
2. API Key 未配置或无效
3. DeepSeek API 服务异常

解决方法：使用本地解析或检查网络和 API Key

### Q: 本地解析准确率低
**A:** 本地解析使用正则表达式，对 PDF 格式要求较高。建议：
1. 使用 AI 解析
2. 确保 PDF 格式规范（题号、选项清晰）

### Q: 题目显示不完整
**A:** 
1. 检查 PDF 文件是否完整
2. 尝试使用不同的解析模式
3. 查看右侧日志了解详细信息
