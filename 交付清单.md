## 🚀 部署说明

### 快速部署步骤

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **配置 API Key**（可选，AI解析需要）
   ```bash
   export DEEPSEEK_API_KEY="your-api-key"
   ```

3. **运行程序**
   ```bash
   # 桌面版
   python exam_app.py
   
   # 网页版
   python server.py
   ```

### 系统要求

- Python 3.8+
- 内存 512MB+
- 磁盘 100MB+
- 网络连接（AI解析需要）
