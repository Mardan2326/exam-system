# PDF试卷模拟考试器 - 轻量版

## 🚀 快速开始

1. **打开** `Exam_Lite.html` 文件
2. **上传** JSON格式的题目文件
3. **开始** 模拟考试

就这么简单！无需安装任何依赖。

## 📝 如何获取JSON题目文件？

### 推荐方法：使用智谱AI官网

1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/)
2. 上传你的PDF试卷
3. 复制以下提示词：

```
请仔细识别这个PDF文件中的所有选择题，并按照以下JSON格式输出：

基础格式（纯文本）：
[
  {
    "id": "1",
    "text": "题干内容",
    "options": {
      "A": "选项A的完整内容",
      "B": "选项B的完整内容", 
      "C": "选项C的完整内容",
      "D": "选项D的完整内容"
    },
    "answer": "A"
  }
]

带图格式（题干/选项包含图片）：
- 题干带图：添加 "image": "<base64编码>" 字段
- 选项带图：{"text": "描述", "image": "<base64>"}

⚠️ 重要要求：
1. 必须提取所有题目，不要遗漏
2. 题号从1开始连续编号
3. **答案处理**：
   - 如果PDF中有标准答案，直接提取到answer字段（A/B/C/D）
   - **如果PDF中没有答案，你需要根据题目内容和选项，运用你的知识判断正确答案**
   - 如果题目过于专业或你完全无法判断，才将answer设为null
4. 保持题目和选项的原文，不要简化
5. 只返回纯JSON数组，不要其他文字
6. **如果题目或选项包含图片**：
   - 题干带图：添加 "image" 字段
   - 选项带图：将选项改为 {"text":"...", "image":"..."} 格式
   - 图片使用base64编码或URL
```

4. 保存AI输出的JSON到文件（如 `questions.json`）
5. 在 `Exam_Lite.html` 中上传该文件
